# SEMMA Digital - Sistema de Gestão de Licenciamento Ambiental

O **SEMMA Digital** é uma plataforma robusta desenvolvida para otimizar a fiscalização e o controle de licenças ambientais do município. 

O sistema transforma o monitoramento passivo em uma gestão ativa e preventiva, focada na regularidade das empresas e na eficiência da arrecadação municipal.

Este projeto faz parte do portfólio de soluções tecnológicas da **KM Engenharia & Tecnologia**.

## Funcionalidades Principais

* **Painel de Gestão Estratégica (Dashboard):** Visualização imediata do total de empresas, licenças vencidas e processos em fase de renovação.
* **Régua de Alerta de 30 Dias:** Sistema inteligente de cores que prioriza automaticamente licenças com vencimento próximo (inferior a 30 dias).
* **Módulo de Notificação WhatsApp:** Automação de envio de mensagens técnicas personalizadas para os proprietários das empresas com apenas um clique.
* **Relatório de Consulta por CNPJ:** Geração de relatórios técnicos prontos para impressão ou PDF, incluindo data/hora da consulta e auditoria.
* **Auditoria de Cadastro:** Registro compulsório do **Nome do Servidor** e **Cargo**, garantindo a rastreabilidade de todas as inserções no banco de dados.
* **Gestão de Dados Segura:** Funcionalidades de exportação e importação de backup (.json) para garantir a integridade das informações.

## Padrões de Licenciamento (SEMMA)

O sistema já vem pré-configurado com os principais padrões utilizados pela secretaria:

* **LP:** Licença Prévia
* **LI:** Licença de Instalação
* **LO:** Licença de Operação
* **Outorga:** Outorga de Uso de Água
* **Autorização:** Autorizações Ambientais Diversas

## Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3 (Bootstrap 5.3)
* **Ícones:** Bootstrap Icons
* **Lógica de Negócio:** JavaScript (ES6+) puro
* **Banco de Dados:** LocalStorage (Persistência no navegador com suporte a backup externo)
